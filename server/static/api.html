API Reference

API Path Response Type Description
/api/currencies/num int The number of unique currencies currently tracked by the server.
/api/currencies/raw JSON A JSON object with raw currency values.
/api/currencies/prediction CSV A CSV with a magical stock prediction, its price, and its percentage increase over the last 24 hours.
/api/currencies/prediction-string string A string with a magical stock prediction.
/api/image/<stock-symbol> RGB565 data An image that shows the graph of a given stock symbol.
